# FloodSense – Web Dashboard

FloodSense is a multidisciplinary senior project at KFUPM focused on urban flash flood monitoring and early warning.

The platform combines IoT sensing, cloud infrastructure, and AI-based flood risk analysis to provide real-time monitoring and visualization through an interactive web dashboard.

Built with:

* Next.js
* Supabase
* ESP32-based sensor nodes
